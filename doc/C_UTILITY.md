# C_UTILITY


**DEFINE_DEFAULT_TYPE_PUBLIC_MEMBERS**
- `@brief` Define an alias (`typedef`) and mandatory functions for a given type. Use `ALIAS` as type in containers.
- `@param` `TYPE`   Type to be aliased.
- `@param` `ALIAS`  Typedef and public name prefix for `TYPE`. (e.g., `ALIAS_create`, etc.).
